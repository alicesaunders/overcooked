
# Recipe Search Engine
<b> overcooked </b>  is a semantic recipe search engine built on the corpus ReciFine (2.2+ million recipes), derived from [RecipeNLG](https://aclanthology.org/2020.inlg-1.4/) , and available on [hugging face.](https://huggingface.co/datasets/nuhuibrahim/recifine)

1.0.0: Users can input available ingredients from a free text field, and then return relevant recipe suggestions. 

v1.1.0: Users can generate a new recipe using a generative model.

<b> Authors</b>: Alice Saunders, Georgia Gunson
